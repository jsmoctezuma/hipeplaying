Guía rápida

Instalación en 4 pasos

Necesitas OBS Studio, Streamer.bot con WebSocket Server activo y la acción importada.

Abre el admin

Entra a admin.html, revisa la conexión y ajusta layout, fuente, escala, animaciones y ocultado.

Importa la acción

Copia el código de importación y pégalo en Streamer.bot para crear la acción de lectura SMTC.

Activa WebSocket

Usa host 127.0.0.1 y puerto 8080, o los datos que tengas configurados.

Pégalo en OBS

Copia la URL generada y úsala como Browser Source con tamaño 1200 × 300.

Después de importar

Qué debe existir

✅Acción: HP - Poll Music.
✅WebSocket Server: activo en Streamer.bot.
✅URL OBS: generada desde el admin con poll=350.
✅Fuente de navegador: 1200 de ancho por 300 de alto.
✅Preview: el admin lo muestra al 50% solo visualmente, sin afectar OBS.
Diseños incluidos

Layouts del reproductor

Los tres estilos están conectados desde el admin y se mandan por URL al widget.

Normal

Diseño original con portada 270x270 y panel derecho con plataforma, estado, título, artista, barra y tiempos.

Compacto

Oculta la portada, elimina el espacio intermedio y expande el panel a todo el ancho útil.

Minimal

Portada más texto limpio: título, artista y tiempo actual, sin fondo/tarjeta en el panel derecho.

Ayuda rápida

Problemas comunes

La mayoría de problemas se resuelven revisando WebSocket, tamaño OBS y si la app realmente entrega datos por SMTC.

No aparece nada

Revisa que Streamer.bot esté abierto, WebSocket activo y que el admin marque Conectado.

OBS se ve mal escalado

Configura la fuente de navegador en 1200 × 300. El preview del admin está al 50% solo para verlo cómodo.

Amazon no muestra duración

Amazon Music a veces no entrega duración/progreso por SMTC. El widget muestra -:-- cuando no hay duración real.

Portadas pixeladas

Algunos navegadores entregan miniaturas pequeñas, por ejemplo 120x120. El backend no recomprime ni baja calidad.

¿Por qué no usar capturas de pantalla?

Porque el widget lee la sesión multimedia de Windows y renderiza su propio diseño, más limpio y configurable.

¿Funciona con navegadores?

Sí, pero como fallback. Las apps dedicadas tienen prioridad para evitar que un video de navegador tape Spotify, Amazon o YouTube Music app.

¿Qué hace “Ocultar después”?

Si está en 0, el widget queda visible mientras haya música. Si pones 10, aparece con canción nueva y se oculta después de 10 segundos.
